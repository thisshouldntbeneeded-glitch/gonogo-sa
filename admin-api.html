Available Endpoints

GET /rest/v1/reviews
Fetch all user reviews. Supports Supabase query filters (eq, like, order, limit).
GET /rest/v1/reviews?brand_name=eq.{brand}&status=eq.approved
Fetch approved reviews for a specific brand.
POST /rest/v1/reviews
Submit a new review. Required fields: brand_name, reviewer_name, review_text. Status defaults to "pending".
// Example body { "brand_name": "Discovery Bank", "reviewer_name": "Jane", "review_text": "Great app experience!", "category_slug": "banking", "status": "pending" }
GET JavaScript: GoNoGoAPI.getAllBrands()
Brand data is served statically from JS files. Use the GoNoGoAPI wrapper to access brands, categories, and scores.
